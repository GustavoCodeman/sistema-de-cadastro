# Sistema de cadastro de jogos

> Status do projeto: Em desenvolvemento

Comandos git:

```
git clone <LINK> = clonar os arquivo do repositorio web.

git clone -branch new_feature <repositorio> = clonar um repositorio a partir de um ramo (banch)

git status = ver os status.

git commit (nome do arquivo)(se quiser atualizar mais de um arquivo coloque só um ponto(.)) -m  "(nome que esta dando)" = salvar arquivo.

-m = mensagem.

git push origin (ramificação) = passar as alterações locais para a web.

git log = mostra quem atualizou e quando.

git log --online = mostra os arquivo e o nome como comitamos ele.

git pull =  é usado para buscar e baixar conteúdo de repositórios remotos e fazer a atualização imediata ao repositório local para que os conteúdos sejam iguais.

git restore --source (numero do commit) (nome do arquivo ou ponto (.) para voltar todos os arquivos) = serve para voltar versão.

git log -p = ver mais detalhado as atualizações.

git log --author="user_name" = ver atualização por nome.

git log --since=1.month.ago --until=1.day.ago = ver atualização por data.

git log --pretty="format:%h %s" = Serve para trazer o hash seguido da mensagem de commit.

git add (nome ou .) = todas as pagina que forem adicionadas, vao ser ouvidas.

git checkout -b (nome da nova ramificaçã) = ele criar uma ramificação

git switch (nome da ramificação) = ir para alguma ramificação

git branch = mostra tadas as ramificações do nosso projeto

git merge (nome da ramificação) = alterar a aplicação para a ramificação principal.

```
